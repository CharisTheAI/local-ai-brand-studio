# Changelog

All notable changes to this project should be documented here from this point forward.

The format is intentionally simple and practical for a fast-moving local product build.

## 0.5.0 - 2026-05-27

First documented pre-1.0 release baseline.

Highlights:

- multi-provider image model selection in the UI
- provider/model registry for OpenAI and Google Gemini adapters
- local-first asset library and saved text asset workflows
- QA audit panel with visible flagged issues and correction status
- hard-repair generation path for critical identity failures
- README expanded with clearer install, setup, and local-use guidance

Notes:

- this is a working pre-1.0 version
- broader testing, fidelity tuning, and more provider-specific refinement are still expected
